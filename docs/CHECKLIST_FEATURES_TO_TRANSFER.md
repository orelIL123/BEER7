# רשימת תוספות להעברה לפרויקט אחר

מסמך זה מכיל פיצ'רים ועדכוני UI להעברה צעד־אחר־צעד לפרויקט אחר. כל בלוק מתאר פיצ'ר אחד עם קבצים רלוונטיים והנחיות.

---

## 1. סידור הלוגו + לוח משרות

### 1.1 סידור הלוגו

- **הדר כללי (`_layout.tsx`):** לוגו + כותרת במרכז, כפתורי התראות/חזור מצד אחד, המבורגר ופרופיל מהצד השני.
- **מסך הבית (`index.tsx`):** בר עליון עם לוגו גדול במרכז (180×90), שמתכהה בגלילה (אנימציה).

### 1.2 לוח משרות

- **מסך ייעודי:** `app/(tabs)/jobs.tsx`
  - Hero ירוק
  - פילטרים: הכל / מלאה / חלקית / סטודנטים
  - רשימת משרות עם כרטיסים
  - כפתור "פרסם משרה"
  - מצב ריק (EmptyState)
- **רישום ב־Tabs:** `Tabs.Screen name="jobs"` עם `href: null` (מסך קיים אבל לא בטאב בר).
- **במסך הבית:** סעיף "משרות בעיר" + באנר ירוק קליקבילי ("מחפש/ת עבודה?" → ניווט ל־community עם `filter: 'job'`).
- **בקהילה:** קטגוריה "עבודה" (`job`), תמיכה ב־`params.filter=job`, EmptyState וצבעים ייעודיים למשרות.

### 1.3 טיפוסים ונתונים

- **CommunityCategory** – כולל `'job'`.
- **CommunityPost** – שדות: `category`, `phone`, `contact`, `location`, `authorName`, `date`.

### 1.4 קבצים רלוונטיים

| קובץ | תיאור |
|------|--------|
| `app/(tabs)/_layout.tsx` | הדר + רישום מסך jobs |
| `app/(tabs)/index.tsx` | לוגו במסך הבית + באנר משרות וסגנונות |
| `app/(tabs)/jobs.tsx` | המסך המלא של לוח משרות |
| `app/(tabs)/community.tsx` | קטגוריית job וקישור מהבית |
| `constants/Types.ts` | CommunityCategory ו־CommunityPost |

**הערה:** בקובץ המלא בפרויקט המקור יש הפניות לשורות, שמות סגנונות והסברים מפורטים – מומלץ להעתיק משם צעד־אחר־צעד.
